
# Tic Tac Toe Game

## Project Description
This is a simple Tic Tac Toe game developed using Python. It is a two-player console-based game where players take turns placing X and O on a 3×3 board.

## Features
- Two-player game
- Win detection
- Draw detection
- Input validation
- Play again option

## Technologies Used
- Python 3
- Visual Studio Code

## How to Run
1. Clone this repository.
2. Open the project in Visual Studio Code.
3. Run the following command:

```bash
python tic_tac_toe.py
```
